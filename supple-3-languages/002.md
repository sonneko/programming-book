# HelloWorld・FizzBazz集

「新しい言語を学ぼう！」と思ったとき、まず真っ先に書くべきなのが **Hello World**（画面にテキストを出す）であり、次に書くべきなのが **FizzBuzz**（基本的な制御構文を網羅する）です。

ここでは、本書で紹介する様々な言語による実装を並べてみます。それぞれの文法の違いを楽しみながら、「何が共通しているか」を見つけてみてください。

## 1. Python
本書の 1 章で学んだ、最もフレンドリーな言語です。

### Hello World
```python
print("Hello World")
```

### FizzBuzz
```python
for i in range(1, 101):
    if i % 15 == 0:
        print("FizzBuzz")
    elif i % 3 == 0:
        print("Fizz")
    elif i % 5 == 0:
        print("Buzz")
    else:
        print(i)
```

## 2. JavaScript (Node.js)
Web の公用語。`console.log` が基本です。

### Hello World
```javascript
console.log("Hello World");
```

### FizzBuzz
```javascript
for (let i = 1; i <= 100; i++) {
  if (i % 15 === 0) console.log("FizzBuzz");
  else if (i % 3 === 0) console.log("Fizz");
  else if (i % 5 === 0) console.log("Buzz");
  else console.log(i);
}
```

## 3. C言語
10 章で学ぶ、低レイヤーの王者。`main` 関数が必要で、`printf` で出力します。

### Hello World
```c
#include <stdio.h>

int main() {
    printf("Hello World\n");
    return 0;
}
```

### FizzBuzz
```c
#include <stdio.h>

int main() {
    for (int i = 1; i <= 100; i++) {
        if (i % 15 == 0) printf("FizzBuzz\n");
        else if (i % 3 == 0) printf("Fizz\n");
        else if (i % 5 == 0) printf("Buzz\n");
        else printf("%d\n", i);
    }
    return 0;
}
```

## 4. Rust
11 章で紹介する、安全性と速度を両立した現代的な言語。`!` がつくのはマクロの印です。

### Hello World
```rust
fn main() {
    println!("Hello World");
}
```

### FizzBuzz
```rust
fn main() {
    for i in 1..=100 {
        match (i % 3, i % 5) {
            (0, 0) => println!("FizzBuzz"),
            (0, _) => println!("Fizz"),
            (_, 0) => println!("Buzz"),
            _      => println!("{}", i),
        }
    }
}
```
※ Rust では `if` 文の代わりに `match` という強力なパターンマッチングを使うのが一般的です。

## 5. Haskell
12 章で学ぶ、純粋関数型言語。ループ（`for`）という概念がなく、再帰やリスト操作で記述します。

### Hello World
```haskell
main = putStrLn "Hello World"
```

### FizzBuzz
```haskell
fizzbuzz i
    | i % 15 == 0 = "FizzBuzz"
    | i % 3 == 0  = "Fizz"
    | i % 5 == 0  = "Buzz"
    | otherwise   = show i

main = mapM_ (putStrLn . fizzbuzz) [1..100]
```

## 文法の違いを観察する
- **セミコロン (`;`)**: JavaScript や C では必須ですが、Python や Haskell にはありません。
- **インデント（字下げ）**: Python ではインデント自体が「意味（ブロック）」を持ちますが、C や JS では `{}` で囲むのが基本です。
- **型の指定**: Rust や C では、変数が何であるかを厳密に意識する必要があります。

## なぜ FizzBuzz なのか？
FizzBuzz は、プログラマとしての「基本的な素養」を測るための簡単なテストとして、かつて有名になりました。
1. **ループ**: 1 から 100 まで動かせるか？
2. **条件分岐**: 3 の倍数、5 の倍数、15 の倍数を正しく判定できるか？
3. **演算**: あまり（`%`）の計算を知っているか？
4. **出力**: 結果を正しく画面に表示できるか？

この 4 要素が含まれているため、新しい言語を学ぶ際の「最初の一歩」として最適なのです。

## まとめ
どうでしたか？言語によって見た目は全く違いますが、やっていること（ロジック）はどれも同じです。
自分の好きな言語を見つけて、その「方言」でコンピュータと対話する楽しさを味わってみてください。
