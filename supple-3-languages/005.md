# Android開発の言語

世界で最も多くのシェアを誇るモバイル OS、**Android**。そのアプリ開発には、Google が推奨する 2 つの主要な言語があります。

かつては **Java** が絶対的な王者でしたが、現在では **Kotlin** という、よりモダンで効率的な言語が Google によって「第一推奨言語（Kotlin First）」と位置づけられています。

## 1. Kotlin (コトリン)
2011 年に JetBrains 社（有名な開発ツールメーカー）によって発表された言語です。

### 特徴：Java をより簡潔に、安全にした現代の旗手
Kotlin は、Java と完全に互換性を保ちつつ、Java の弱点を克服するために設計されました。コード量が大幅に削減され、Swift と同様に `NullPointerException`（プログラムをクラッシュさせる最大の原因）を未然に防ぐ仕組みが組み込まれています。

```kotlin
// Kotlin の Hello World
fun main() {
    println("Hello World")
}

// FizzBuzz
fun main() {
    for (i in 1..100) {
        when {
            i % 15 == 0 -> println("FizzBuzz")
            i % 3 == 0 -> println("Fizz")
            i % 5 == 0 -> println("Buzz")
            else -> println(i)
        }
    }
}
```
Java に比べて非常にスッキリしており、Python や JavaScript を学んだ後でも馴染みやすい文法です。

### 主な用途：Android 開発のデファクトスタンダード
現在、Android アプリの新規開発のほとんどは Kotlin で行われています。Google の公式ドキュメントやサンプルコードも、まずは Kotlin で提供されるようになっています。

## 2. Java (ジャバ)
Android 誕生の瞬間から、長らく開発を支えてきた巨塔です。

### 特徴：厳格、重厚、かつ堅牢
Java は非常に厳格なオブジェクト指向言語です。「何をするにもクラス（Class）が必要」という特徴があり、初心者には少しコードが長く（冗長に）感じられるかもしれません。

```java
// Java の Hello World
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
```
しかし、その厳格さゆえに、大規模な開発でもバグを防ぎやすく、世界中の企業システムで採用されている信頼感があります。

## 開発に必要なもの：Android Studio
Android アプリを開発するには、Google が提供している **Android Studio** という開発ツールを使います。Mac はもちろん、Windows や Linux でも開発が可能です。Apple 開発と違って、ハードウェアの制約が少ないのは Android 開発の大きなメリットの一つです。

## JVM（Java 仮想マシン）という魔法
Kotlin も Java も、実は共通の土台の上で動いています。それが **JVM (Java Virtual Machine)** です。
プログラムを CPU が直接理解できる機械語にするのではなく、一度「中間コード」にコンパイルし、それを JVM という通訳者が実行します。

これにより、「一度書けば、どこでも動く（Write Once, Run Anywhere）」という Java の理想が実現されています。あなたが書いた Kotlin のコードも、裏側では Java のコードと同じように JVM の命令に変換されて Android 上で動いているのです。

## マルチプラットフォームへの広がり
最近では **Kotlin Multiplatform (KMP)** という技術が登場し、Android だけでなく iOS や Web の開発も一つの Kotlin コードで共有できるようになりつつあります。この「横断的な開発」は、今後のアプリ開発のトレンドとして非常に注目されています。

## まとめ
Android 開発は、オープンで多様性に富んだ世界です。
「まずは手堅く、業界標準の Java を触っておく」のも良いですが、これから新しく始めるなら、ぜひ **Kotlin** の扉を叩いてみてください。その書きやすさと、モダンな機能の数々に驚くはずです。

Android スマホを手に、自分の書いたコードで画面を動かす。その瞬間、あなたはただのスマホユーザーから、モバイル社会を創造するエンジニアへと変貌を遂げるのです。
