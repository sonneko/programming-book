# .NET系言語

Microsoft が開発し、Windows の世界を中心に発展してきた **.NET（ドットネット）**。
かつては Windows 専用の技術でしたが、現在では **.NET 8**（およびそれ以降）として完全にオープンソース化され、Mac でも Linux でも、そして iOS や Android でも動く、世界最強クラスの開発プラットフォームへと進化しました。

その中心に位置するのが、美しさと実用性を兼ね備えた言語、**C#（シーシャープ）** です。

## 1. C# (シーシャープ)
2000 年に登場した、Microsoft の看板言語です。

### 特徴：Java のライバルであり、究極の優等生
C# は、Java と非常によく似た文法を持っていますが、Java が保守的であるのに対し、C# は積極的に新しい機能をどんどん取り入れてきました。その結果、「Java よりも書きやすく、Python よりも高速で、Rust よりも親しみやすい」という、非の打ち所がない万能言語へと成長しました。

```csharp
// C# の Hello World (最新の書き方)
using System;

Console.WriteLine("Hello World");

// FizzBuzz
for (int i = 1; i <= 100; i++)
{
    var output = (i % 3 == 0, i % 5 == 0) switch
    {
        (true, true) => "FizzBuzz",
        (true, _)    => "Fizz",
        (_, true)    => "Buzz",
        _            => i.ToString()
    };
    Console.WriteLine(output);
}
```
`switch` 式（パターンマッチング）など、非常にモダンで洗練された記述が可能です。

### 主な用途：ゲーム、Web、エンタープライズ
- **ゲーム開発**: 世界で最も使われているゲームエンジン **Unity** の開発言語は C# です。『ポケモン GO』や『原神』も、C# で作られています。
- **Web 開発**: ASP.NET Core というフレームワークを使い、超高速な Web サーバーを構築できます。
- **デスクトップアプリ**: Windows のソフトを作るなら、今でも C# が第一選択肢です。

## 2. Visual Basic (.NET)
かつて、初心者向けのプログラミング言語として一世を風靡した言語です。

### 特徴：英語に近い読みやすさ
`If...Then...End If` のように、記号よりも単語を多用する文法が特徴です。現在は C# に主役を譲っていますが、Excel のマクロ（VBA）などにその面影を強く残しています。

## 3. F# (エフシャープ)
12 章で学ぶ「関数型プログラミング」の思想を .NET に持ち込んだ言語です。

### 特徴：データの変換に特化した、数学的な美しさ
`|>`（パイプライン演算子）を使ってデータを次々と加工していくスタイルが得意で、データ分析や金融システムなどで高く評価されています。

## .NET の共通土台：中間言語（IL）
Java が JVM 上で動くのと同様に、C# や F# も **CLR (Common Language Runtime)** という共通の土台の上で動きます。
コンパイルされると「共通中間言語 (CIL)」という形になり、それが実行時に CPU ごとの機械語に変換されます。

この仕組みのおかげで、C# で書いたライブラリを F# から呼び出す、といった言語をまたいだ連携が非常にスムーズに行えます。

## まとめ
.NET 系言語、特に **C#** を学ぶことは、エンジニアとしてのキャリアにおいて非常に強力な「安定資産」になります。
「Windows アプリを作りたい」という人はもちろん、「Unity でゲームを作りたい」「本格的な Web 開発をしたい」という人にとっても、C# は最高に満足度の高い選択肢となるでしょう。

Microsoft という巨大企業のバックアップがありつつ、コミュニティによってオープンに進化し続ける。そんな .NET の世界は、これからプログラミングを志す皆さんを、広い懐で迎え入れてくれます。
