# SQL文2

データを条件付きで操作する方法を学びます。

### 条件操作

- **WHERE**: 条件に一致するデータを絞り込む。
- **UPDATE**: データを更新する。
- **DELETE**: データを削除する。

```sql
UPDATE users SET name = 'Bob' WHERE id = 1;
DELETE FROM users WHERE id = 1;
```

条件を指定せずに更新や削除を行うと全てのデータが影響を受けるため、`WHERE`句の使用は必須のスキルです。

### 知識の拡充

- **技術的深み**: `UPDATE`や`DELETE`時には、データの一貫性を保つためにトランザクションログ（WALなど）への書き込みが発生します。
- **低レイヤーの現実**: データの物理的な削除（`DELETE`）は即座にディスク容量を解放しないことが多く、VACUUM処理が必要になります。
- **設計思想**: データの冪等性と一貫性を保証するため、意図しない変更を防ぐための慎重なクエリ設計が求められます。
