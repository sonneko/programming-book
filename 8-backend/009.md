# SQL文1

SQL（Structured Query Language）は、DBを操作するための標準言語です。まずはデータの作成と取得から始めます。

### 基本操作

- **CREATE TABLE**: テーブルを作成する。
- **INSERT**: データを挿入する。
- **SELECT**: データを取得する。

```sql
CREATE TABLE users (id INTEGER PRIMARY KEY, name TEXT);
INSERT INTO users (name) VALUES ('Alice');
SELECT * FROM users;
```

これらの基本的なSQL文を理解することが、データ駆動型アプリケーション開発の第一歩です。

### 知識の拡充

- **技術的深み**: SQLパーサーがクエリを抽象構文木（AST）に変換し、クエリプランナーが最適な実行計画を生成します。
- **低レイヤーの現実**: `INSERT`はディスクのページ単位で書き込まれるため、頻繁な挿入はディスクI/O負荷を増大させます。
- **設計思想**: 宣言的な言語仕様により、データの物理的な格納方法を意識せずにビジネスロジックに集中できる設計です。
