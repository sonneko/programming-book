# GeminiAPI

AI の進化は止まるところを知りません。Google が提供する最新の AI モデル **Gemini** は、その高い知能を **Gemini API** を通じて私たちのプログラムから利用することができます。

Gemini API を使えば、テキストの生成、要約、翻訳、プログラムコードの作成、さらには画像や動画の解析まで、驚くほど簡単に実装できてしまいます。かつては数ヶ月かかっていたような高度な機能を、数行のコードで実現できるようになったのです。これはエンジニアにとって、魔法のような時代と言えるでしょう。

## Gemini API の特徴
Gemini API には、いくつかのモデル（サイズのバリエーション）があります。
- **Gemini 1.5 Pro**: 高度な推論が可能で、非常に長いコンテキスト（一度に扱える情報量）を扱える最高性能モデル。
- **Gemini 1.5 Flash**: 軽量で高速。大量のリクエストを処理する場合や、レスポンスの速さが求められる用途に最適。

特筆すべきは、Gemini 1.5 シリーズが持つ **「100万トークンを超える長いコンテキスト窓」** です。これは、数百ページの本、あるいは数時間の動画を一度に AI に読み込ませ、その内容について質問できることを意味します。他の AI モデルとは一線を画す、Gemini の最大の強みです。

## API キーの取得：Google AI Studio
Gemini API を使い始めるのは非常に簡単です。
[Google AI Studio](https://aistudio.google.com/) にアクセスし、Google アカウントでログインするだけで、API キーを即座に発行できます。

現在、**「Free of charge（無料枠）」** が非常に充実しており、個人の開発やテスト目的であれば、かなりの回数を無料で利用することができます。ただし、無料枠では入力したデータが Google のモデル改善のために利用される可能性があるため、機密情報を入力する際は注意が必要です。

## プロンプトエンジニアリング：AI への指示の出し方
API を通じて AI に指示を出すテキストのことを **プロンプト** と呼びます。AI から期待通りの出力を得るためには、このプロンプトを工夫することが非常に重要です。これを **プロンプトエンジニアリング** と呼びます。

良いプロンプトのポイントは以下の通りです。
1. **役割（ロール）を与える**: 「あなたは優秀な Python プログラマです」といった設定。
2. **具体例（Few-shot）を示す**: 「入力: A → 出力: B」のように、期待する形式の例をいくつか見せる。
3. **制約を設ける**: 「100文字以内で回答してください」「JSON 形式だけで出力してください」など。

## コードによる実装例
Python を使う場合、公式のライブラリ（`google-generativeai`）を使うことで、数行で AI との対話を実現できます。

```python
import google.generativeai as genai

# APIキーの設定
genai.configure(api_key="YOUR_GEMINI_API_KEY")

# モデルの初期化
model = genai.GenerativeModel('gemini-1.5-flash')

# テキスト生成
response = model.generate_content("プログラミングを始めたばかりの学生に、応援メッセージを書いてください。")

print(response.text)
```

これだけで、AI が生成した心温まるメッセージが返ってきます。画像についても、画像データと一緒に「この画像には何が写っていますか？」というテキストを送るだけで解析が可能です。

## システム命令（System Instruction）
最近のアップデートで、モデルに対して「常に守るべきルール」を定義できる **System Instruction** 機能が追加されました。これを使うことで、ユーザーの入力にかかわらず、常に特定の口調で話させたり、特定の形式で出力させたりといった制御がより確実に行えるようになりました。

## Gemini API で広がる世界
- **AI チューター**: 分からないコードを貼り付けると、優しく解説してくれるボット。
- **自動要約ツール**: 長い記事や YouTube 動画の内容を、3 行でまとめてくれるツール。
- **クリエイティブ支援**: あなたが書いた小説の続きを一緒に考えたり、キャラクターのイラスト案をテキストで生成したりする。
- **マルチモーダル連携**: スマートフォンのカメラで撮った冷蔵庫の中身から、作れるレシピを提案するアプリ。

## まとめ
Gemini API は、あなたのプログラムに「脳」を与えるようなものです。これまでのプログラミングは、すべての条件分岐（if 文）を人間が書く必要がありました。しかし AI を組み込むことで、「なんとなくの意味」を理解し、曖昧な入力に対しても賢く応答するシステムが作れるようになります。

AI 時代において、AI を「使う側」になるか「使われる側」になるかの境界線は、こうした API を自らの手で動かしてみたかどうかにあります。まずは Google AI Studio で、AI との対話をプログラムから制御する感覚を掴んでみてください。
