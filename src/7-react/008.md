# ReactRouterとは

React Routerは、Reactアプリケーションにおいてページルーティングを実現するための標準的なライブラリです。

### 基本機能

URLに基づいて表示するコンポーネントを切り替えることで、SPAにおける画面遷移を実現します。

```jsx
<Routes>
  <Route path="/" element={<Home />} />
  <Route path="/about" element={<About />} />
</Routes>
```

React Routerを利用することで、複雑なアプリケーションでもURLと画面状態を同期させることが可能になります。
