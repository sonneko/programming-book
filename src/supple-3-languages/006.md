# Java言語

世界で最も有名なプログラミング言語の一つ、**Java（ジャバ）**。
「名前は聞いたことがある」「コーヒーのマークのやつでしょ？」という人も多いはずです。Java は 1995 年の誕生以来、四半世紀以上にわたってエンジニアの世界で絶対的な地位を築いてきました。

前章の「Android 開発」でも触れましたが、Java は単なるモバイル用言語ではありません。むしろ、銀行のシステムや大企業の基幹業務など、**「絶対に止まってはいけない、極めて高い信頼性が求められる場所」** で真価を発揮する言語です。

## Java の最大の特徴：厳格なオブジェクト指向
Java を一言で表すなら「ルールに厳しい、真面目な言語」です。
Python や JavaScript では適当に書いても動いてしまうようなコードも、Java では「型が違う！」「クラスが定義されていない！」と厳しく叱られます。

```java
// Java の FizzBuzz
public class FizzBuzz {
    public static void main(String[] args) {
        for (int i = 1; i <= 100; i++) {
            if (i % 15 == 0) {
                System.out.println("FizzBuzz");
            } else if (i % 3 == 0) {
                System.out.println("Fizz");
            } else if (i % 5 == 0) {
                System.out.println("Buzz");
            } else {
                System.out.println(i);
            }
        }
    }
}
```
コードを見ると、`public class` や `public static void main` といった「おまじない」のような長い記述が必要であることが分かります。これは、プログラムのすべての要素を「オブジェクト（物）」として厳格に管理しようとする、Java の設計思想の現れです。

## なぜ Java はこれほど普及したのか？
1. **プラットフォーム非依存**: 「JVM（Java 仮想マシン）」のおかげで、同じコードが Windows でも Mac でも、はたまた巨大なスーパーコンピュータの上でも全く同じように動きます。
2. **豊富なライブラリ**: 歴史が長いため、世界中のエンジニアが作った「便利な部品（ライブラリ）」が膨大に存在します。何かを作ろうとしたとき、ゼロから作る必要はほとんどありません。
3. **静的型付けによる安全性**: コンパイル時に多くのミスを見つけてくれるため、数千人規模で一つのプログラムを作るような巨大なプロジェクトでも、品質を保つことができます。

## 「Java」と「JavaScript」は全くの別物
ここで初心者が最も混乱するポイントを整理しておきます。
**「Java と JavaScript は、メロンとメロンパンくらい違います。」**

- **Java**: コンパイルが必要。主にサーバー側や Android で動く。厳格。
- **JavaScript**: ブラウザ上で動く。動的型付け。柔軟。

名前が似ているのは、JavaScript が生まれた当時、Java があまりにも有名だったため、その人気にあやかって（マーケティング目的で）名付けられたという歴史的経緯があります。文法こそ少し似ていますが、その設計思想や動く場所は全くの別物です。

## 今、Java を学ぶ意味
「最近は Python や Rust の方が流行っているんじゃないの？」と思うかもしれません。確かに、新しい流行は常にあります。しかし、世界中の膨大なシステムが今でも Java で動いており、それらをメンテナンスし、進化させるための Java エンジニアの需要は、今後何十年もなくなることはないでしょう。

また、Java を学ぶことは、プログラミングの王道である **「オブジェクト指向」** の考え方を徹底的に身につけることに繋がります。Java で培った「厳格な設計力」は、他のどんな言語を学ぶ際にも必ずあなたの助けになります。

## まとめ
Java は、現代社会を裏側から支える「巨大なインフラ」のような言語です。
派手なアニメーションを作ったり、AI で絵を描いたりするのは他の言語に任せ、Java は黙々と、正確に、そして頑丈にプログラムを動かし続けます。

「エンジニアとして一生モノの、揺るぎない基礎体力をつけたい」。そう思うなら、Java という高い壁に挑戦してみる価値は十分にあります。
