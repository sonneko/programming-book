# 関数定義

JavaScriptにおける関数は「第一級オブジェクト」であり、変数に代入したり、他の関数の引数として渡したり、関数から関数を返したりすることが可能です。

### 関数宣言と関数式

関数を定義するには、主に「関数宣言」と「関数式」の2つの方法があります。

```javascript
// 関数宣言 (巻き上げが発生する)
function sayHello() {
  console.log('Hello');
}

// 関数式 (変数に代入するため、巻き上げは発生しない)
const sayHi = function() {
  console.log('Hi');
};
```

### アロー関数 (Arrow Function)

ES6で導入されたアロー関数は、より簡潔な記法で関数を定義できます。

```javascript
const add = (a, b) => a + b;
```

アロー関数は `this` の扱いが通常の関数とは異なり、定義した場所のスコープを継承するため、コールバック関数などで非常に便利です。現在のJavaScript開発ではアロー関数が主流となっています。
