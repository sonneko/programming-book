# コレクション型

JavaScriptにおいて、複数のデータをまとめて扱うためのデータ構造を「コレクション」と呼びます。主に配列（Array）とオブジェクト（Object）が使われますが、ES6以降は `Map` や `Set` といった特殊なコレクションも標準で利用可能です。

### 配列 (Array)

配列は、順序付けられたデータの集合です。インデックス（0から始まる番号）を使って要素にアクセスします。

```javascript
const numbers = [1, 2, 3, 4, 5];
console.log(numbers[0]); // 1
```

### オブジェクト (Object)

オブジェクトは、キーと値のペアを保持する集合です。順序はなく、キーを使って値にアクセスします。

```javascript
const user = {
  name: 'Alice',
  age: 25
};
console.log(user.name); // Alice
```

### Map と Set

- **Map**: キーと値のペアを保持します。オブジェクトと似ていますが、キーにオブジェクトそのものを含めあらゆるデータ型を使用でき、要素の挿入順序が保持されます。
- **Set**: 重複のない値の集合を保持します。配列から重複を取り除きたい場合などに便利です。

これらを適切に使い分けることで、データの効率的な管理が可能になります。
