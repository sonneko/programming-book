# 演習

これまでの知識を定着させるために、実際にコードを書いてみましょう。

### 課題

1. **配列の操作**: 1から10までの数値が入った配列を作成し、`filter`メソッドを使って偶数のみを抽出した新しい配列を作成してください。
2. **オブジェクトの作成**: ユーザーの情報を保持するオブジェクトを作成し、名前を変更するメソッドを追加してください。
3. **DOM操作**: ページ上にボタンを作成し、クリックされたらテキストの内容を「Clicked!」に変更するスクリプトを書いてください。

### ヒント

- 配列のメソッド (`map`, `filter`, `reduce`) を駆使することで、手続き的なループよりも宣言的にコードを書くことができます。
- DOM操作では、`addEventListener` を使用してイベントを監視します。

これらの課題を完了することで、JavaScriptの基本的な文法とデータ操作の感覚を掴むことができます。
