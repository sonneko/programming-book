# シンボルの記法

シンボル（Symbol）は、ES6で導入されたプリミティブ型です。一意で変更不可能な値を作成するために使用されます。

### なぜシンボルを使うのか

オブジェクトのキーとして文字列を使用すると、他のコードが同じキー名を上書きしてしまうリスクがあります。シンボルを使えば、絶対に衝突しない一意のキーを作成できます。

```javascript
const mySymbol = Symbol('description');
const obj = {
  [mySymbol]: 'value'
};
```

### 特徴

- `Symbol()` で作成されるシンボルは、毎回必ず一意になります。
- オブジェクトのプロパティの隠蔽や、内部的なメタデータの管理によく使われます。

シンボルは日常的な開発で頻繁に使うわけではありませんが、ライブラリ開発や、意図しないデータの上書きを防ぎたい複雑なデータ構造においては強力なツールとなります。
