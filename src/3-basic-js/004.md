# 基本の制御構造

プログラムの流れを制御することは、開発の基礎です。JavaScriptでは、条件分岐とループ処理を用いて処理の実行順序を柔軟に制御します。

### 条件分岐

最も基本的な条件分岐は `if` 文です。特定の条件が真（true）である場合にのみ、ブロック内のコードが実行されます。

```javascript
const age = 20;

if (age >= 18) {
  console.log('成人です');
} else {
  console.log('未成年です');
}
```

`if-else if-else` を組み合わせることで、多段階の条件分岐も可能です。

### ループ処理

同じ処理を繰り返す必要がある場合、ループを使います。

- **`for`文**: 回数が決まっている繰り返しに適しています。
- **`while`文**: 条件が満たされている間、繰り返し処理を行います。

```javascript
// for文の例
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

これらの制御構造を適切に組み合わせることで、複雑なロジックを構成します。
