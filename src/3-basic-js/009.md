# オブジェクト型

オブジェクト型は、JavaScriptにおいてデータを構造化するための最も重要なデータ構造です。キーと値のペアを保持し、プロパティやメソッドを持つことができます。

### オブジェクトの定義とアクセス

オブジェクトは `{}` を使って定義します。

```javascript
const car = {
  brand: 'Toyota',
  model: 'Corolla',
  drive() {
    console.log('Driving...');
  }
};

console.log(car.brand); // ドット記法
console.log(car['model']); // ブラケット記法
```

### プロトタイプベースの継承

JavaScriptのオブジェクトは「プロトタイプチェーン」という仕組みによって継承を実現しています。現代では `class` 構文を使って直感的にオブジェクト指向プログラミングを行うのが一般的ですが、その裏側ではプロトタイプが動作しています。

```javascript
class Vehicle {
  constructor(name) {
    this.name = name;
  }
}
```

オブジェクトは単なるデータの箱ではなく、振る舞い（メソッド）と状態（プロパティ）をカプセル化する単位として理解しましょう。
