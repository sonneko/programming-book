# switch文

`switch` 文は、一つの変数の値に基づいて、複数のケース（分岐）を扱うための制御構造です。`if-else if` を連ねるよりも、特定の値を比較する場合には可読性が高くなる場合があります。

### 基本的な構文

```javascript
const fruit = 'apple';

switch (fruit) {
  case 'apple':
    console.log('りんごです');
    break;
  case 'banana':
    console.log('バナナです');
    break;
  default:
    console.log('その他の果物です');
}
```

### 重要な注意点：break

`switch` 文では、各 `case` の後に `break` を記述することが必須です（設計上の意図がある場合を除き）。`break` を忘れると、次の `case` の処理まで実行されてしまう「フォールスルー」と呼ばれる挙動が発生します。

最近のJavaScript開発では、`switch` 文よりもオブジェクトやマップを利用したマッピングのほうが柔軟で安全な場合も多いです。用途に応じて使い分けましょう。
