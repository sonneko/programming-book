# DOM操作

DOM（Document Object Model）は、HTMLドキュメントをプログラムから操作するためのインタフェースです。JavaScriptを用いることで、Webページの内容やスタイルを動的に変更できます。

### DOMツリーの取得

ブラウザ上の要素にアクセスするには、主に以下のメソッドを使用します。

```javascript
const element = document.getElementById('my-id');
const elements = document.querySelectorAll('.my-class');
```

### 要素の操作

取得した要素に対して、プロパティやスタイルを変更します。

```javascript
element.textContent = 'Updated content';
element.style.color = 'red';
element.classList.add('active');
```

DOM操作はブラウザのレンダリングエンジンに対して重い負荷をかける可能性があるため、最小限に抑えるのがパフォーマンス向上のコツです。Reactなどのライブラリが仮想DOMを用いてこれを効率化しているのは、このためです。
