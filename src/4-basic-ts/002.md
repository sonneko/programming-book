# npmの使い方

npm（Node Package Manager）は、Node.jsの標準パッケージマネージャです。膨大な数のライブラリを簡単にインストール・管理できるため、現代のJavaScript開発では必須です。

### 基本的な操作

パッケージをインストールするには、`npm install <package-name>` を使用します。

```bash
# パッケージのインストール
npm install axios
```

### package.jsonの管理

`package.json` はプロジェクトの依存関係を管理する重要なファイルです。

- `dependencies`: 本番環境で必要なライブラリ。
- `devDependencies`: 開発時（テストやビルド）にのみ必要なライブラリ。

```bash
# devDependenciesに追加
npm install typescript --save-dev
```

npmを活用することで、コードの再利用性を高め、効率的な開発環境を構築できます。
