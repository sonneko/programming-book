# オブジェクト指向の用語整理

オブジェクト指向を正しく理解し、設計するための基本的な用語を整理します。

- **クラス (Class)**: オブジェクトの設計図。
- **インスタンス (Instance)**: 設計図（クラス）から生成された実体。
- **プロパティ (Property)**: オブジェクトが持つ状態（変数）。
- **メソッド (Method)**: オブジェクトが持つ振る舞い（関数）。
- **継承 (Inheritance)**: 既存のクラスの機能を引き継いで新しいクラスを作成すること。
- **ポリモーフィズム (Polymorphism)**: 同じ名前のメソッドを異なるオブジェクトに対して呼び出し、異なる結果を得ること。
- **インターフェース (Interface)**: オブジェクトが従うべき契約。

これらの用語は、チーム開発において設計上の対話を行う際に共通言語となります。
