```python
scores = [85, 92, 78, 90, 66, 89, 74, 95, 82, 80]

# ここにコードを書いてください
```

<details>
<summary>ヒント</summary>

- 合計を保存する変数（最初は0）を作り、`for`文でリストの中身を一つずつ足していきます。
- 平均は、`合計 / 要素数` で求められます。要素数は `len(scores)` で取得できます。
</details>
