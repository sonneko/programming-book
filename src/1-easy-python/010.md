# for文

```python
for index in range(10):
    print(str(index))
```

for文もwhileと同じく繰り返しを表す構文の一つです。例えば上記のプログラムは以下のようなプログラムと同じ挙動を示します。

```python
index = 0
print(str(index))
index = 1
print(str(index))
index = 2
print(str(index))
# ...(中略)
index = 9
print(str(index))
```

range(10)は、0から9までの数字の羅列を「オンデマンドで」[^note1]生成する「イテレータ」という値を返すという関数です。このfor文の後ろ(`range(10)`)には、イテレータという種類の値ならなんでも入ります。
イテレータとは、値を次々と作る「もの」のことです。
`range`関数がイテレータなのではなく、`range`関数が返す「もの」がイテレータであるということに注意してください。

イテレータについてはまだ理解できなくても大丈夫です。なかなか最初は難しい概念ですし、また勉強することになりますので。

[^note1]: 「オンデマンドで」というのは「（事前に作るのではなく）必要になった瞬間に」という意味です。プログラミングの文脈では、「遅延評価」も同じ意味です。

例えば、次に学ぶリストもイテレータの一種であるのでここに入れることが可能です。

```python
fruits = ["apple", "banana", "orange"] # リスト型

for fruit in fruits:
    print(fruit) # apple banana orange
```
