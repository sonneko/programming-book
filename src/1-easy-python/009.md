# while文

print関数を100回呼び出したい時を考えます。
脳筋にprintを100回書くこともできますが、やっぱり1000回にしてと言われた時、はたまた、繰り返しを行う回数が計算してみないといけない場合に対応できません。
よってそのための構文を紹介します。

```python
while True:
    print("Hello!")
```

`while文`は中の処理を繰り返すという構文です。後ろの値が`True`である間繰り返し実行されます。この場合は無限ループを起こすので実行が止まりません。

一番最初の例のように100回繰り返したい場合は以下のようにします。

```python
index = 0

while index < 100:
    print("hello!" + str(index) + "times.")
    index = index + 1
```

頭の中で一つずつシミュレーションしてみてから、実際に実行してみてください。

この場合は、`print`関数は`index`という変数が`0`から`99`の間実行されます。

コンピュータの世界では、数字の始まりは0です。なので、n個の数字が必要なら、数字はn-1までになります。ややこしく思うかも知れませんが、そのうちこっちの方が自然になってしまうぐらいには慣れますのでお気になさらず。
