# 真偽値

今回は真偽値をより深掘りしていきます。

典型的に中学校の数学で習う論理演算をプログラミングの世界でも使うことができます。

例えば、`x`が10から20の範囲にあるかどうかを判定するにはどうすればいいでしょうか？

```python
x = 14
if (x > 10):
    if (x < 20):
        print("x is between 10 and 20.")
```

このようなこともできますが、以下のようなこともできます。

```python
x = 14
if (x > 10 and x < 20):
    print("x is between 10 and 20.")
```

数学のように`10 < x < 20`と書くこともできます。Pythonはこの書き方をサポートしている珍しい言語の一つです。

```python
x = 14
if 10 < x < 20:
    print("x is between 10 and 20.")
```

しかし、基本的には`and`を使って以下のように書くのが一般的です。なぜなら、他のほとんどのプログラミング言語では上の書き方はできないからです。

```python
x = 14
if x > 10 and x < 20:
    print("x is between 10 and 20.")
```

`x > 10`の結果と、`x < 20`の結果が`and`で結ばれることで、どちらも`True`である時のみ最終的な結果が`True`になるようにできます。

なぜ他の言語で`10 < x < 20`と書けないかというと、多くの言語では「左から順番に」計算されるからです。
つまり、まず`10 < x`が計算されて真偽値（True/False）になり、次に`[真偽値] < 20`という、数と真偽値を比べるというおかしな計算になってしまうからです。Pythonではこれを特別に数学と同じように扱えるようにしてくれています。

他にもどちらかが`True`ならばもう一方がどうであっても`True`になるという`or`も使うことができます。

```python
x = 32
output = x < 10 or x > 20
print(output) # False
```

この例では、10未満もしくは20より大きいことを確認しています。

Pythonでは、`and`の代わりに`&&`や、`or`の代わりに`||`も使うことができます。他のプログラミング言語だと後者しか使えないことが多いので注意してください。
