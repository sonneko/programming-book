# Three.js入門

Three.jsは、ブラウザ上で3Dグラフィックスを表示するための強力なライブラリです。WebGLを直接扱うのは非常に難易度が高いですが、Three.jsはその複雑さをラップし、直感的なAPIを提供します。

### Three.jsの基本概念

- **Scene (シーン)**: 3Dオブジェクトを配置する空間。
- **Camera (カメラ)**: シーンをどの視点から見るかを決定。
- **Renderer (レンダラー)**: シーンとカメラをもとに、最終的な画像を描画。

### 簡単な描画

```javascript
const scene = new THREE.Scene();
const camera = new THREE.PerspectiveCamera(75, window.innerWidth / window.innerHeight, 0.1, 1000);
const renderer = new THREE.WebGLRenderer();
renderer.setSize(window.innerWidth, window.innerHeight);
document.body.appendChild(renderer.domElement);
```

Three.jsは、Webブラウザを単なる文書閲覧ツールではなく、リッチな体験を提供するプラットフォームへと変貌させます。
