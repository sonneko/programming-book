# よく使うhtmlタグ

HTMLには数百種類のタグがありますが、実際に日常的に使うものは限られています。まずは以下の「主要なタグ」を覚えましょう。

### テキスト関連

-   `<h1>` 〜 `<h6>`: 見出し。`h1`が最も重要。
-   `<p>`: 段落 (Paragraph)。
-   `<a>`: リンク (Anchor)。`href`属性で飛び先を指定。
-   `<span>`: 文中の特定の部分だけ囲みたいときに使う。

### 構造・グループ化

-   `<div>`: 特に意味を持たない「箱」。レイアウトのために要素をまとめるのに最も多用されます。
-   `<ul>` / `<li>`: 箇条書き。
-   `<header>`, `<main>`, `<footer>`: サイトの大きな区切り。

### コンテンツ挿入

-   `<img>`: 画像。`src`属性で画像パス、`alt`属性で説明。
-   `<button>`: ボタン。

### インプット（フォーム）

-   `<input>`: 文字入力欄など。
-   `<form>`: 入力データをサーバーへ送るためのまとまり。

### セマンティックHTMLの意識

何でもかんでも `<div>` で囲む（通称：div地獄）ことを避けたほうがよいでしょう。

```html
<!-- 悪い例 -->
<div class="header">見出し</div>

<!-- 良い例 -->
<header>見出し</header>
```

ブラウザや検索エンジン、音声読み上げソフトなどに対して、「ここはヘッダーですよ」「ここはメインコンテンツですよ」という **意味 (Semantics)** を正しく伝えることで、アクセシビリティやSEO（検索エンジン最適化）が向上します。
