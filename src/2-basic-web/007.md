# VisualStudioCodeの導入

プログラミングはメモ帳でもできますが、効率を劇的に上げるために **エディタ** を使いましょう。現在、世界で最も使われているのが Microsoftの開発した **Visual Studio Code (VS Code)** です。

### なぜ VS Code なのか？

1.  **シンタックスハイライト**: コードに色がつき、読みやすくなる。
2.  **補完機能**: 途中まで打つと候補を出してくれる（ミスが減る）。
3.  **拡張機能**: 自分の好きな機能を追加できる。
4.  **内蔵ターミナル**: 黒い画面（ターミナル）をエディタ内で開ける。

### Web制作に必須の拡張機能

VS Codeをインストールしたら、以下の拡張機能を導入することをお勧めします。

-   **Japanese Language Pack**: UIを日本語化します。
-   **Live Server**: 保存するたびにブラウザが自動更新され、制作が爆速になります。
-   **Prettier**: コードの見た目を自動で綺麗に整えてくれます。

### 環境構築という「最初の一歩」

プログラミングを始めた人の多くが、最初の「環境構築」で挫折します。しかし、自分の道具を使いやすく整えることは、職人がカンナを研ぐのと同じです。一度整えてしまえば、あとの学習がずっと楽になります。
