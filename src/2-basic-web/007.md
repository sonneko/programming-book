# VisualStudioCodeの導入

プログラミングは、極論すれば「メモ帳」でも書くことができます。しかし、世界中のプロのエンジニアがメモ帳でコードを書くことはありません。なぜなら、開発を劇的に速く、正確にし、何より「楽しく」してくれる最高の相棒が存在するからです。

その相棒が、Microsoft が開発した **Visual Studio Code (VS Code)** です。

私が初めて VS Code を使ったとき、今まで 1 時間かかっていたタイピングやデバッグが、わずか数分で終わるようになった衝撃は今でも忘れられません。ここでは、その導入と Web 制作を爆速にするための設定をお話しします。

## なぜ VS Code が選ばれるのか？

現在、世界で最も使われているコードエディタは VS Code です。その理由は、単に「無料だから」だけではありません。

1. **シンタックスハイライト**:
   コードの種類に応じて色がつきます。タグの閉じ忘れやタイポ（スペルミス）が一目でわかるようになります。
2. **インテリセンス（自動補完）**:
   `bac` と打てば `background-color` を候補に出してくれます。全てのプロパティを覚える必要がなくなり、入力ミスもゼロになります。
3. **拡張機能の豊富さ**:
   自分が必要な機能をスマートフォンにアプリを入れるような感覚で追加できます。
4. **内蔵ターミナル**:
   黒い画面（コマンドライン）をエディタ内で直接操作できます。これは後の章で Git を使うときに非常に便利です。

## Web 制作に必須の「神」拡張機能 3 選

VS Code をインストールしたら、まずは左側のアイコン（四角いブロックのマーク）から、以下の 3 つを検索してインストールしましょう。

### 1. Live Server
HTML を保存するたびに、ブラウザが自動でリロード（更新）されます。
いちいちブラウザに戻って「F5」キーを押す必要がなくなります。この快適さを知ると、もう元には戻れません。

### 2. Prettier - Code formatter
コードの見た目を自動で綺麗に整えてくれます。
インデント（字下げ）がバラバラでも、保存した瞬間にプロのような整ったコードに修正されます。「見た目の美しさはバグの少なさに直結する」と私は信じています。

### 3. Japanese Language Pack for VS Code
VS Code のメニューを日本語化します。英語に自信がある人以外は、最初に入れておきましょう。

## 低レイヤーの視点：エディタとパフォーマンス

「VS Code って重くない？」と感じる人もいるかもしれません。
実は VS Code は、**Electron** という技術で作られています。これは「Web ブラウザ（Chromium）上で動くデスクトップアプリ」のようなものです。

そのため、メモ帳のようなネイティブアプリ（C++ などで作られたもの）に比べると、多くのメモリ（RAM）を消費します。しかし、現代の PC なら全く問題にならないレベルですし、その代償として「Web の技術を使って誰でも簡単に拡張機能を作れる」という最強のメリットを手に入れました。
「メモリを贅沢に使って、人間の開発時間を短縮する」。これが現代のソフトウェア開発の基本的な考え方の一つです。

## 私の視点：ショートカットキーを「指」で覚える

エディタを使いこなすコツは、マウスを捨ててショートカットキーを使うことです。

- `Ctrl + S` (保存)
- `Ctrl + C` / `V` (コピー＆ペースト)
- `Ctrl + /` (コメントアウト：コードを一瞬で無効化する)

これらを「指が勝手に動く」レベルまで練習してください。タイピング速度が上がると、思考のスピードを邪魔されずにプログラミングに没頭できるようになります。それはまるで、コンピュータと直接脳がつながっているような、最高の体験ですよ。

### まとめ

- 開発には「メモ帳」ではなく、専用のエディタ（VS Code）を使おう。
- 自動補完と色付けが、ミスを減らし開発を加速させる。
- 「Live Server」と「Prettier」は Web 制作の必須アイテム。
- VS Code はメモリを消費するが、それ以上の利便性がある。
- ショートカットキーを使いこなして、思考を加速させよう。

### 展望

プロの道具を手に入れたら、次は CSS の表現力をさらに引き出す「アニメーション」の世界を覗いてみましょう。静的なページが生き生きと動き出す瞬間は、何度見ても楽しいものです。
