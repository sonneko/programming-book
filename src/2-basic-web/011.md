# 演習：自己紹介ページを作ってみよう

2章で学んだ知識を総動員して、簡単な自己紹介ページを作成してみましょう。

### 要件

1.  `index.html` と `style.css` の2つのファイルを作成する。
2.  HTMLには以下の要素を含める。
    -   名前（`<h1>`）
    -   自分の趣味や目標（`<p>`）
    -   好きなサイトへのリンク（`<a>`）
    -   画像（`<img>` - 何かフリー素材のURLなど）
3.  CSSで以下の装飾を行う。
    -   背景色を変える。
    -   全体の幅を制限して、画面の中央に寄せる（`margin: 0 auto;` を調べてみよう）。
    -   フォントを読みやすく変える。

### ヒント：検証ツールを使おう

ブラウザ（Chromeなど）で自分のページを開き、右クリックして「検証」を選ぶと、**デベロッパーツール** が開きます。

ここでCSSをリアルタイムに書き換えて、見た目がどう変わるかを試行錯誤することができます。これはプロのエンジニアが毎日、数千回と使う最も重要なデバッグ手法です。

---

完成したら、ブラウザで正しく表示されるか確認してください。
もし余裕があれば、Flexboxを使って、画像と文章を横に並べるレイアウトにも挑戦してみましょう。
