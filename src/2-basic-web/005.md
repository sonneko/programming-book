# cssセレクター

セレクターは、CSSを適用するターゲットを特定するための強力な道具です。

### 基本的なセレクター

1.  **要素セレクター**: タグ名で指定。
    ```css
    p { color: gray; }
    ```
2.  **クラスセレクター**: HTMLで `class="name"` と書いたものに指定。先頭に `.` を付ける。最も多用します。
    ```css
    .main-text { font-weight: bold; }
    ```
3.  **IDセレクター**: HTMLで `id="name"` と書いたものに指定。先頭に `#` を付ける。ページ内に1つだけ。
    ```css
    #unique-button { background: red; }
    ```

### 組み合わせ・階層

セレクターを組み合わせることで、より詳細なターゲットを狙えます。

-   **子孫セレクター**: `A B` (Aの中にあるBすべて)
-   **子セレクター**: `A > B` (Aの直下にあるBだけ)
-   **複数指定**: `A, B` (AとBの両方)

### 擬似クラス (Pseudo-classes)

「マウスが乗ったとき」や「1番目の要素」など、特定の状態を指定できます。

```css
button:hover {
  background: orange;
}

li:first-child {
  color: blue;
}
```
