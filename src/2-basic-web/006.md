# よく使うcssプロパティ

CSSプロパティは膨大な数がありますが、まずはレイアウトと文字装飾に関する基本をマスターしましょう。

### ボックスモデル

CSSの最も重要な概念は、すべての要素が「箱（ボックス）」であるということです。

1.  **width / height**: 箱の幅と高さ。
2.  **margin**: 箱の外側の余白。
3.  **padding**: 箱の内側の余白。
4.  **border**: 箱の枠線。

これらの数値をピクセル単位で微調整していくのが、コーディングの基本作業になります。

### 文字装飾

-   **color**: 文字の色。
-   **font-size**: 文字の大きさ。
-   **font-family**: フォントの種類（ゴシック体、明朝体など）。
-   **text-align**: 文字の揃え（left, center, right）。

### 色と背景

-   **background-color**: 背景色。
-   **opacity**: 不透明度（0〜1）。

### 単位の使い分け

初心者はすべて `px`（ピクセル）で書きがちですが、シニアエンジニアは状況に応じて単位を使い分けます。

-   **px**: 固定値。
-   **%**: 親要素に対する割合。
-   **rem / em**: フォントサイズを基準とした相対値。
-   **vw / vh**: 画面の幅・高さに対する割合。

特に `rem` や `%` を適切に使うことで、スマートフォンのような画面サイズの異なる端末でも綺麗に見える「レスポンシブデザイン」が実現しやすくなります。
