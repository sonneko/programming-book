# 演習：自己紹介ページを作ってみよう

ここまでの 2 章で、Web の仕組みから HTML の骨組み、CSS の装飾、そして Flexbox や Grid を使ったモダンなレイアウトまで、一気に駆け抜けてきました。

知識は、使って初めて「スキル」に変わります。今回は、これまでの学びを総動員して、あなたの **「世界でたった一つの自己紹介ページ」** を作り上げましょう。

## 今回のミッション

以下の要件を満たす Web ページを、ゼロから作成してください。

### 1. 開発環境の準備
- VS Code を開き、新しいフォルダを作成します。
- `index.html` と `style.css` の 2 つのファイルを作成します。
- `index.html` の `head` 内で `style.css` を正しく読み込みます。
- **Live Server** を起動して、リアルタイムで確認できるようにします。

### 2. HTML 構造（セマンティックに！）
- **Header**: あなたのサイトのタイトル（`<h1>`）と、簡単なメニュー（`<ul>`）を入れます。
- **Main**:
  - あなたの写真（またはイラスト）を表示します（`<img>`）。
  - 自己紹介文（`<h2>` と `<p>`）を書きます。
  - 好きな技術や趣味を箇条書きでリストアップします。
- **Footer**: コピーライト（`© 2024 Your Name`）を入れます。

### 3. CSS 装飾（モダンに！）
- **Google Fonts** などを調べて、好きなフォントを適用してみましょう。
- 背景色や文字色を、自分のイメージに合わせて設定します。
- **Flexbox** を使って、ヘッダーのタイトルとメニューを横に並べてください。
- 写真と文章を、大きな画面では横並びに、スマホ（画面を細くしたとき）では縦並びになるように調整してみましょう（メディアクエリに挑戦！）。

## ヒント：ブラウザは「最高の教科書」

制作中、「なぜか色が反映されない」「要素が横に並ばない」といったトラブルは必ず起きます。そんな時は、迷わずブラウザの **「デベロッパーツール（検証ツール）」** を開きましょう。

- 右クリック ＞ 「検証」を選択。
- 左上の矢印アイコンで、気になる要素をクリック。
- 右側の「Styles」パネルで、どの CSS が当たっているか、上書きされていないかを確認します。

プロのエンジニアは、10 行コードを書くたびに 1 回はここを見て、試行錯誤しています。デベロッパーツールを使いこなすことは、コーディングができることと同じくらい重要です。

大きな画像は、それだけでページの読み込み速度（**LCP**：*Largest Contentful Paint*）を悪化させ、ユーザーの離脱を招きます。
- 画像の縦横サイズを、実際に表示する大きさに合わせてリサイズする。
- `WebP` などの高圧縮なフォーマットに変換する。
- サーバーからのレスポンスを速くするために、画像を軽量化する。

「自分の PC で見えているから OK」ではなく、世界中の様々なネットワーク環境にいるユーザーを想像して、データを最適化する。それが「優しい」エンジニアのあり方です。

## 私の視点：完璧より「公開」を

最初からプロが作ったような洗練されたデザインを目指す必要はありません。
「HTML で意味を伝え、CSS で最低限の読みやすさを整える」。まずはそこからスタートしましょう。

そして、できればこのページを **GitHub Pages** や **Vercel** といったサービスを使って、インターネット上に「公開」してみてください。
自分の書いたコードが、URL ひとつで世界中の誰からでも見られるようになる。その瞬間の興奮は、プログラミングを続ける上での最高のご褒美になりますよ。

### まとめ

- VS Code と Live Server を駆使して、効率よくコーディングする。
- `Header`, `Main`, `Footer` といったセマンティックなタグで構造化する。
- Flexbox を使って、要素を自在に配置する。
- デベロッパーツール（検証）は、迷った時の最強の味方。
- 画像の軽量化など、ユーザーの利便性（パフォーマンス）を意識する。

### 展望

お疲れ様でした！ 2 章を完遂したことで、あなたは「静的な Web サイト」を作れるようになりました。
しかし、Web の世界はこれだけでは終わりません。次は、この静かなサイトに「魂」を吹き込み、ユーザーの操作に反応してダイナミックに変化させる魔法の杖―― **JavaScript** の世界へ足を踏み入れましょう。
